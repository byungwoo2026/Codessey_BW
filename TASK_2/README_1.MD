# 🧠 퀴즈 게임 프로그램

파이썬으로 만든 객관식 4지선다 퀴즈 게임입니다.
퀴즈 풀기, 추가, 목록 확인, 점수 확인 기능을 제공하며,
프로그램을 종료해도 데이터가 유지됩니다.

## 🎮 기능 설명

### 1. 퀴즈 풀기 (메뉴 1번)
- 등록된 퀴즈를 **랜덤 순서**로 출제합니다.
- 4개의 선택지 중 **번호(1~4)**를 입력하여 정답을 맞춥니다.
- 풀이 후 **점수와 정답률**을 보여줍니다.
- **최고 점수**를 자동으로 갱신하고 저장합니다.

```
📝 문제 1/5
📌 파이썬에서 화면에 출력할 때 사용하는 함수는?
----------------------------------------
  1. input()
  2. print()
  3. echo()
  4. write()
----------------------------------------
정답 번호를 입력하세요 (1~4): 2
✅ 정답입니다! 🎉
```

### 2. 퀴즈 추가 (메뉴 2번)
- **문제, 선택지 4개, 정답 번호**를 입력하여 새 퀴즈를 추가합니다.
- 추가된 퀴즈는 **즉시 JSON 파일에 저장**됩니다.

```
문제를 입력하세요: 파이썬의 자료형이 아닌 것은?
선택지 4개를 입력하세요:
  1번 선택지: int
  2번 선택지: float
  3번 선택지: char
  4번 선택지: str
정답 번호를 입력하세요 (1~4): 3
✅ 퀴즈가 추가되었습니다!
```

### 3. 퀴즈 목록 (메뉴 3번)
- 등록된 **모든 퀴즈**를 보여줍니다.
- 각 퀴즈의 **문제, 선택지, 정답**을 확인할 수 있습니다.

```
[1] 파이썬에서 화면에 출력할 때 사용하는 함수는?
    ✔ 정답: 2번 (print())
```

### 4. 점수 확인 (메뉴 4번)
- **최고 점수**와 **정답률**을 확인할 수 있습니다.
- 최고 점수는 프로그램 종료 후에도 유지됩니다.

### 5. 종료 (메뉴 5번)
- 데이터를 **저장하고** 프로그램을 종료합니다.

## 🏗️ 코드 구조

### 클래스 구성

```
main.py
├── Quiz 클래스        ← 퀴즈 1개를 표현
│   ├── __init__()     : 문제, 선택지, 정답 초기화
│   ├── display()      : 퀴즈 출력
│   ├── check_answer() : 정답 확인
│   ├── to_dict()      : 딕셔너리로 변환 (저장용)
│   └── from_dict()    : 딕셔너리에서 복원 (불러오기용)
│
├── QuizGame 클래스    ← 게임 전체를 관리
│   ├── __init__()     : 초기화 및 데이터 불러오기
│   ├── play_quiz()    : 퀴즈 풀기
│   ├── add_quiz()     : 퀴즈 추가
│   ├── view_quizzes() : 퀴즈 목록 보기
│   ├── view_score()   : 점수 확인
│   ├── save_state()   : JSON 파일에 저장
│   ├── load_state()   : JSON 파일에서 불러오기
│   ├── show_menu()    : 메뉴 출력
│   └── run()          : 메인 루프
│
└── 프로그램 시작점
    └── if __name__ == "__main__"
```

### Quiz 클래스 속성

| 속성 | 타입 | 설명 | 예시 |
|------|------|------|------|
| question | str | 문제 | "파이썬에서 출력 함수는?" |
| choices | list | 선택지 4개 | ["input()", "print()", "echo()", "write()"] |
| answer | int | 정답 번호 (1~4) | 2 |

### 데이터 저장 형식 (state.json)

```json
{
  "quizzes": [
    {
      "question": "파이썬에서 화면에 출력할 때 사용하는 함수는?",
      "choices": ["input()", "print()", "echo()", "write()"],
      "answer": 2
    }
  ],
  "high_score": 3
}
```

## ⚠️ 주의사항

1. **state.json 파일**은 프로그램 첫 실행 시 자동 생성됩니다.
2. state.json을 직접 수정하면 **데이터가 손상**될 수 있습니다.
3. state.json을 삭제하면 **기본 퀴즈 5개로 초기화**됩니다.
4. 정답 번호는 반드시 **1~4 사이의 정수**로 입력해야 합니다.

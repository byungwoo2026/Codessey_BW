# 본과정 서버 에이전트 운영 과제

## 1. 개요
안정적인 에이전트 운영을 위해 서버 보안 설정, 네트워크 구성 및 자동화된 모니터링 시스템을 구축하였습니다.

## 2. 주요 수행 내역
* **SSH 보안 강화**: 포트를 **20022**로 변경하고 root 접속을 차단함.
* **방화벽 설정**: **UFW**를 활성화하여 20022(SSH) 및 15034(App) 포트만 허용함.
* **에이전트 구성**: `agent-admin`, `agent-dev`, `agent-test` 환경 분리 및 공통 모듈 배치.
* **자동화**: `crontab`을 통해 `monitor.sh`를 **매분(1분 주기)** 실행하도록 등록함.

## 3. 모니터링 스크립트 (monitor.sh) 규정
* **프로세스 감시**: `agent-dev`, `agent-core` 생존 확인.
* **포트 감시**: **15034/tcp** LISTEN 상태 확인.
* **리소스 경고 임계치**:
  - CPU 사용률 > 80%
  - Memory 사용률 > 10%
  - Disk 사용률 > 80%

## 4. 증적 로그 경로
* 로그 위치: `/var/log/agent-app/monitor.log`
